# WhatsApp AI Agent for Indian SMBs

## Product: WhatsApp Assistant for Small Businesses

**Core Value**: Every Indian SMB gets an AI receptionist that never sleeps, never takes leave, and costs less than a human.

## Features (MVP)

1. **AI Auto-Reply**: Instantly reply to customer queries 24/7
2. **FAQ Bot**: Pre-configured answers to common questions
3. **Order Taking**: Simple product/service ordering via chat
4. **Appointment Booking**: Calendar integration for bookings
5. **Multi-language**: Hindi, English, regional languages

## Pricing

- **Free**: 50 messages/month
- **Pro (₹999/mo)**: Unlimited messages, 5 products, basic analytics
- **Business (₹2999/mo)**: Unlimited everything, custom training, priority support

## Tech Stack

- Backend: FastAPI (Python)
- WhatsApp: Twilio API
- AI: Local models (GLM-5 via Ollama) or external API
- Deployment: Railway/Render/Vercel
